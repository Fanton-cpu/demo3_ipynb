## Instalación

- Clonar el proyecto con git clone (pagina del proyecto).
- Crear un entorno virtual (python -m venv env) 
- Instalar las librerias del requirements.txt (las que use en el proyecto) 
{import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
